Overview
Designed and simulated a 3-transistor Pass Transistor Logic (PTL) NAND gate using LTspice to study reduced transistor count and low-power digital logic implementation.
Tool Used
LTspice
Features
PTL-based NAND gate design
Reduced transistor implementation
Logic verification through simulation waveforms
Simulation
Circuit operation verified using LTspice transient analysis.
